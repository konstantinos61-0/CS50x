## Week 1 — C
An introduction to the C programming language and the Linux CLI. The first two problems (mario and me) are very simple introductory material.
### Credit
A program in C that checks the validity of a given credit card number, using the Luhn's Algorithm to calculate its "checksum". 
It takes as input a credit card number and outputs whether its a valid American Express, MasterCard or Visa credit card number.  
**Usage**: `./credit`
